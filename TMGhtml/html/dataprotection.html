Условия защиты данных

Таллиннская средняя школа Мустйыэ руководствуется правовыми нормами, регулирующими обработку персональных данных.

Условия защиты данных города Таллинна

Мы обрабатываем ваши персональные данные (например, имя и фамилию, код социального страхования, дату рождения, адрес проживания, контактную информацию и т. д.) следующим образом:

  • При исполнении общественного долга,
  • Во исполнение юридического обязательства,
  • Во исполнение заключенного с вами договора.

    • Нам нужно согласие:

  • Опубликовать имя, фото, видео или творческую работу учащегося на школьном веб-сайте, информационных экранах или в школьных социальных сетях. Цель обработки персональных данных – отражение школьной жизни.
  • При добавлении адреса электронной почты родителей в список родителей класса, в целях облегчения связи классного руководителя, учителей и руководства школы с родителями и для быстрой передачи информации, школа создает список адресов электронной почты родителей каждого класса.
  • Школа не передает ваши персональные данные третьим лицам, за исключением случаев, когда это требуется по закону или если это необходимо для предоставления вам услуг по поддержке учебы (питание, электронный студенческий билет, электронный дневник).
    • Согласие на обработку персональных данных (начальная школа)
    Согласие на обработку персональных данных (гимназия)

    Обязанности школьного специалиста по защите данных выполняет специалист по защите данных канцелярии городского секретаря Таллиннской городской управы.С городским специалистом по защите данных можно связаться по адресу электронной почты: andmekaitse@tallinnlv.ee

    Контактным лицом школы по защите данных, задачей которого является координация решения вопросов защиты данных школы, является секретарь Кристина Степанова, адрес электронной почты: andmekaitse@mjg.ee

    Подробный обзор обработки данных учреждения можно получить в Таллиннском регистре обработки данных.

    После того, как вы вошли в регистр, нажмите кнопку «поиск», затем кнопку «выбрать учреждение», в открывшемся выше поле напишите название учреждения и нажмите кнопку «поиск», затем нажмите название учреждения. учреждение в открывшемся списке (название учреждения выделено красным цветом), наконец нажмите синюю кнопку «поиск». После этого откроется список всех процессов обработки данных учреждения. Более подробное описание каждой обработки данных вы можете просмотреть, нажав кнопку «Просмотр».

    Городской регистр обработки данных находится по адресу: https://atr.tallinn.ee/.
